Installation Instructions

Dexy is written in pure Python, and uses Setuptools. Dexy should be very simple to install. Many of Dexy's filters depend on having other software installed, so depending on how you want to use Dexy you might need to install some extra software too. For example, if you want to use the latex filter to turn documents written in LaTeX into PDFs, then you'll need to install TeXLive or a similar package.

If you are worried about messing up your system, then you can install Dexy and any software you want to use with it within a Virtual Machine. Using a virtual machine is also a good option if your system comes with an old version of Python. There is *very* limited support for Windows.

Dexy should work on Python 2.6 or Python 2.7. There is no support for Python 3.0 yet.

If you run into issues with this install process then let me know.

Easy Install

The recommended way to install Dexy is using easy_install (which is part of setuptools).

{{ d['easy-install.sh|pyg'] }}

or if you are on Python 2.6:

{{ d['easy-install-py26.sh|pyg'] }}

(Dexy WILL NOT WORK on Python 2.6 unless you have installed 'ordereddict' and this does this for you)

If you want an absolutely minimal installation of Dexy without some of the common Python libraries that make Dexy more useful out-of-the-box then you can just do:

{{ d['easy-install-min.sh|pyg'] }}

You can upgrade your Dexy by calling easy_install with the --upgrade flag. There are various other tools available to help with Python package management that you might also use, instead of or in addition to easy_install. Check out yolk, pip,and virtualenv.

Windows

There is currently very limited support for Dexy on Windows. The core of Dexy should work on Windows (as of Version 0.1.7), but many of the Dexy filters will only work on Mac/Linux. Unless you are an experienced developer and are willing to get your hands dirty, write new filters and deal with bugs, to get Dexy to work, then you should set up a Linux virtual machine and run Dexy this way instead.
