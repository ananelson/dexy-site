The 'sed' filter will run sed commands on input text. To start with, you write a .sed file containing the commands you wish to run:

{{ d['example.sed'] }}

Then you prepare a text file with the content you would like to transform with sed:

{{ d['input.txt'] }}

You need to set the text file as an input, you can see how this is done in the config file below (see the Python input filter example for how to run multiple inputs).

Here is the output when run:

{{ d['example.sed|sed'] }}

If you have a really short script, then you might want to create a virtual file and just enter the contents in the .dexy config file below. This is faster, the down side is that it's a little harder to include your original sed script in your document, although it's still possible:

{{ a['virtual.sed|sed'].input_text() }}

You get the same result from a virtual file:

{{ d['virtual.sed|sed'] }}
