Pygments Example

Pygments is a syntax highlighting library. For now it's the only one implemented for Dexy, as it has an amazing number of supported languages and a very solid parser so it's very robust, especially with complex situations like Ruby code embedded in a Ragel parser. Of course more syntax highlighters will be added to Dexy in the future and if you need a particular one now then shout (or DIY and send us a patch).
