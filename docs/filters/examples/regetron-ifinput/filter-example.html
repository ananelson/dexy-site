The regetron filter runs text through regetron and compares it with regex files. You apply the regetron filter to a .regex file, and any files you specify as inputs provide the text that is evaluated against the regex. You can use the ifinput feature to easily match up regexes with corresponding text files, or you can just specify inputs as individual files or wildcards.
