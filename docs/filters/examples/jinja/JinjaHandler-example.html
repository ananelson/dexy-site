Jinja Handler

Jinja is a very important Dexy filter. Jinja is a templating framework, which is what allows you to create dynamic documents in Dexy. Eventually Dexy will support several templating frameworks (like mako, Erb, etc. and many others), but to start with we have chosen Jinja while we work out the best way to work with templating.

Here is a simple text file with some jinja tags.

{{ d['example.txt|dexy'] }}

Here is this same text file after jinja processing.

{{ d['example.txt|jinja'] }}
