Stdout

The stdout handler lets you run code and captures just the output of this code. Many other handlers subclass the stdout handler to take advantage of this functionality. The stdout handler happens to serve as the Python output handler too.

You can pass command line arguments to the stdout handler which will be passed to the command line process. See the erlang handler for an example.
