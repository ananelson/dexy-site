WordPress

WordPress is the most well-supported blogging platform at the moment, it's used for Dexy's own blog. This handler should work with both self-hosted WordPress and wordpress.com blogs. For syntax highlighting, you can either add a stylesheet to your blog's theme, or you can use css inlining which doesn't require a custom stylesheet. For examples of how to use this, look at the dexy blog source.
